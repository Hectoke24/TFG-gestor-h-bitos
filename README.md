# TFG - Gestor de Hábitos

Proyecto de Trabajo de Fin de Grado orientado al desarrollo de una aplicación web para la gestión de hábitos diarios.

## Descripción

La aplicación permitirá al usuario crear, visualizar y gestionar hábitos personales de forma sencilla.
En esta fase inicial del proyecto ya se ha preparado una base funcional con frontend y backend separados.

Actualmente, el proyecto permite:

* mostrar una lista de hábitos
* obtener los hábitos desde el backend
* añadir nuevos hábitos desde el frontend
* mantener una estructura organizada para seguir ampliando funcionalidades

## Objetivo del proyecto

Desarrollar una aplicación web intuitiva que ayude a los usuarios a mejorar su organización personal mediante el seguimiento de hábitos.

## Estado actual

El proyecto se encuentra en una fase inicial de desarrollo.

Hasta el momento se ha realizado:

* creación de la estructura general del proyecto
* configuración del frontend con React y Vite
* configuración del backend con Node.js y Express
* conexión básica entre frontend y backend
* implementación inicial del listado de hábitos
* implementación inicial para añadir hábitos
